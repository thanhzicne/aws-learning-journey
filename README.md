## 🚀 Run the Website Locally

### Requirements

- Hugo Extended
- Git

### Clone Repository

```bash
git clone https://github.com/your-username/your-repository.git
cd your-repository
```

### Start Hugo Server

```bash
hugo server -D
```

Open the local URL displayed in the terminal, usually:

```text
http://localhost:1313
```
