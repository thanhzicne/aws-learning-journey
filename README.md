## Run the Project Locally

### Requirements

- Hugo Extended
- Git

### Clone Repository

```bash
git clone https://github.com/thanhzicne/aws-learning-journey.git
cd aws-learning-journey
```

### Start Hugo Server

```bash
hugo server -D
```

Open your browser and access:

```text
http://localhost:1313
```
